# Privacy Policy — Hotstar Subtitle Styler

**Last updated:** March 18, 2026

## Overview

Hotstar Subtitle Styler is a Chrome extension that customizes the appearance of subtitles on JioHotstar (hotstar.com). It is designed with user privacy as a core principle.

## Data Collection

This extension does **not** collect, transmit, store, or share any user data. Specifically:

- **No personal data** is collected (name, email, account info, etc.)
- **No browsing history** is collected or tracked
- **No analytics or telemetry** is sent to any server
- **No cookies** are set or read by this extension
- **No network requests** are made by this extension

## Data Storage

The extension stores only your subtitle style preferences (font size, color, background opacity, etc.) using Chrome's built-in `chrome.storage.sync` API. This data:

- Is stored locally in your browser and synced across your Chrome devices via your Google account
- Contains only style settings (no personal information)
- Can be cleared at any time by uninstalling the extension or clicking "Reset Defaults"

## Host Permissions

The extension requires access to `*.hotstar.com` and `*.jiohotstar.com` solely to inject CSS styles that modify subtitle appearance on those sites. No data is read from or extracted from these pages.

## Third-Party Services

This extension does not use any third-party services, SDKs, or APIs.

## Changes to This Policy

Any changes to this privacy policy will be reflected in this document with an updated date.
